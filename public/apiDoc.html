API Documentation

Overview

This API provides endpoints to access product information based on language, reference, and category. It allows users to retrieve product data in various languages.

Base URL

The base URL for accessing the API is https://digital.wearekingly.com/api.

Endpoints

Get All Products

URL: /:language

Method: GET

Description: Retrieves all products available in the specified language.

Parameters:

language (required): The language code for which products are to be fetched.

Response:

200 OK: Returns a JSON array containing all products in the specified language.

Example:

GET https://digital.wearekingly.com/api/en

Get Single Product

URL: /:language/ref/:ref

Method: GET

Description: Retrieves information about a single product based on its reference for the specified language.

Parameters:

language (required): The language code for which the product is to be fetched.
ref (required): The reference code of the product.

Response:

200 OK: Returns a JSON object containing details of the specified product.

Example:

GET https://digital.wearekingly.com/api/en/ref/KS04

Get Products by Category

URL: /:language/cat/:cat

Method: GET

Description: Retrieves products belonging to the specified category in the given language.

Parameters:

language (required): The language code for which products are to be fetched.
cat (required): The category of products to be retrieved. Available categories are: socks, compactTextiles, packaging, knitwear, and towels.

Response:

200 OK: Returns a JSON array containing products belonging to the specified category.

Example:

GET https://digital.wearekingly.com/api/en/cat/socks

Language Support

This API supports multiple languages for product information. The language can be specified in the request path using language codes such as en (English), es (Spanish), fr (French), it (Italian), de (German).

Response Format

The API returns responses in JSON format.

Error Handling

400 Bad Request: Invalid request parameters or missing required parameters.

404 Not Found: The requested resource was not found.

500 Internal Server Error: An unexpected error occurred on the server.

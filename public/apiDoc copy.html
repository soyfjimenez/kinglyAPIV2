Kingly Digital Suite API Documentation

Introduction

Welcome to the Kingly Digital Suite (KDS) API documentation. The KDS API provides access to a centralized repository of product references, facilitating consistent and updated information sharing with our customers. The API is designed to be versatile and user-friendly, catering to various product categories.

Base URL

The base URL for accessing the KDS API is:

https://digital.wearekingly.com/api/

Language Support

The API supports multiple languages for data retrieval. You can specify the desired language using the following path:

  • /en/ for English
  • /es/ for Spanish
  • /fr/ for French
  • /it/ for Italian
  • /de/ for German

Note: Prices may vary based on the country, with a 5% increase or no increase. Ensure to use the appropriate language path based on your requirements.

Category Endpoints

Retrieve products in the following categories:

  • /socks
  • /compactTextiles
  • /packaging
  • /knitwear
  • /towels

Example Usage

Retrieve Socks in English:

GET https://digital.wearekingly.com/api/en/socks

Retrieve Compact Textiles in French:

GET https://digital.wearekingly.com/api/fr/compactTextiles

Retrieve the reference KS04 in German:

GET https://digital.wearekingly.com/api/de/ref/KS04

Response Format

The API response provides detailed product information, including title, description, composition, pricing, and weight.

[ { "ref": "ProductReferenceCode", "productTitle": "Product Title", "productDesc": "Product Description", // Additional language-specific fields "cat": "Product Category", "composition": "Product Composition", "prices": "{\"Quantity1\": Price1, \"Quantity2\": Price2, ...}", "weight": ProductWeight } ]

Conclusion
